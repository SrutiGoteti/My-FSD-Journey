# CSS Properties

### Colors Properties

- background-color 
- color
- [Link for color pallete](https://colorhunt.co/)

### Font Properties

- Font-size
  1. Pixel representation(px) - static
  2. Point representation(pt) - static
  3. Em representation(em) - 100% of the parent element - relative
  4. Rem representation(rem) - relative to the root element(HTML)

- Font-family - [Link to fonts](https://fonts.google.com/)

- Font-width
  1. Keywords - Normal, bold
  2. Relative to parent - lighter(-100), bolder(+100)
  3. number - 100 to 900

### Box properties(Layout)

- Height - px, %
- Width - px, %
- Border - [thickness][styling - solid,dashed etc..][color]
- **Note** - when a border is added, height and width of the element does not change
- to set the border for a particular side, we can do that using border-top, border-down, etc..
- to set the border on all sides - [top, right, down, left]
- to set for only two - [top + down, left + right]
- padding - space between element and its border(inner spacing)
- margin - space outside the border(outer spacing that separates elements)
