# Intermediate HTML

## List Element

- Types:
  1. ul(unordered list) (numbered) (order does'nt matter) (ingredients etc..)
  2. ol(ordred list) (bullet points) (order does matter) (instructions etc..)
  3. Nested lists(li in li)
- li are list elements used inside these lists

## Anchor Elements

- Allows to create hyperlinks
- Attributes: add additional element to an HTML element
- Types of attributes:
  1. Global attributes - Applicable to all HTML elements(Ex: draggable etc..)
  2. Local attributes - Only specific to certain HTML elements(Ex: href for anchor tag)
- Adding start attribute to anchor elements, we will be able to number from a specific number
- Adding reverse attribute, the numbering will get reversed

## Image Element

- Has a local attribute called src, whose value will be the location of the image.
- It has a self closing tag, and therefore, is a void element.
- To test the image elements positioning and styling when you dont have your own image, you can get random images from a website called
  [Picsum photos](https://picsum.photos/200) (200 specifies the size of you image)
 - Has another local attribute to support accessebility(screen readers) called alt, that specifies the alternate name of the image that appears if the image is not visible and can be read out
