## The Internet

### Severs

Special Computing machines that serve you all the data and files that you would need to access and interact with certain websites on the internet.

### Clients

Any computer that a user would use to communicate with the server to interact with the internet.

### How do the server and the client communicate with each other?

So, to answer this question, every computer that is connected to the internet has an IP address (kinda like a postal coode for a computer). 

Also, every computer is connected to the internet via an ISP(Internet Service Provider). Ex: BSNL, Airtel etc..

So to access a particular website, let's say "Google", your computer(the client/browser) would send a request through the domain name of that website, in this case(wwww.google.com) to the ISP.

The ISP then sends this request across to the DNS(Domain Name System) Server. This Server is essentially like a phonebook, consisting of all the domain names of websites, as the name suggests, as well as their IP Address as key value pairs.

So, it will find the IP address of Google.com and send it accross the ISP which finally reaches your computer.

Your computer will then make a direct request to that IP address, and the computer that lives at that IP address which is the google servers, will send you back all the data and files you would need to view the google homepage.

So, this is essentially what happens under the hood when the server and the client/browser communicate with each other.

## How do Websites work?

A website is a combination of three essential code files
1. HTML
2. CSS
3. JS

### HTML
The code file resposible for all the contents in your website. Ex: text, forms, buttons etc..

### CSS 
The code files responsible for styling these contents. Ex: Background color, Styling of buttons(rounded corners) etc..

### JS
Enables the website to have functionality and makes them dynamic for the user to interact with. Ex: Send an email etc..
