### HyperText - Hyper Links
### Markup Language - Marksup the content(bold, italic etc..)

### Tags
- Within angular brackets
- enclose the content
- Types - opening and closing tags

### Elements - Tags along with the content inside them.

## Headings

- Only till h6
- Not good practice to have more than 1 h1

## Paragraph

- Separated paragraphs with a line in b/w
- Very useful for screen readers to skip forth to the next paragraphs

## Void Elements

- Elements that do not allow any content inside the tags
- Ex: hr (horizontal rule), br(break)
- Do not use void element in paragraphs unless completely necessary, as this is not a good terms in terms of accessebility.
