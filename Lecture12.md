# Bootstrp

- CSS Framework that consists of pre-made CSS files that you can include in your projects.
- open-source
- other popular CSS frameworks include: Foundation, mui, tailwind etc..
- CSS frameworks are easy, fast, amd consistent styling across platforms, and browser compatability.
- However, it leads to class bloat, where you have lots of styling going to the actual HTML file and therefore, there is no separation between structure and style, so it looks lot less clean.
- Not customizable to personal needs.
- we can get the link tags from CDN(Content Delivery Network)
- Bootstrap is all about multiple cols in a row in a container and how they are laid out
- Bootsrap uses a 12 col layout

## Responsive Layout
- xxl> xl> lg> md> sm screens
- depending on the screen sizes above, we can layout our cols.
- Ex: col-lg-6
