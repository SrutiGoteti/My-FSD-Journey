# HTML Boiler Plate

## `<!DOCTYPE html>`

- Specifies which version of HTML the file was written in
- Latest version of HTML is HTML5

## `<html lang = "en"></html>`

- The language attribute lang is so that the screen reader knows which language to read in

## `<head></head>`

- Information here is not going to be displayed to our user.
- Should consist of the meta tag - `<meta charset = "UTF-8">`. This ensures that the characters get rendered correctly.
- Should consist of viewport attribute
- Should consist of the title tag - `<title></title>`

## `<body></body>`

- All the contents go here that the user can see
